# UnifyVault Integration & E2E Tests (`tests/`)

This directory contains end-to-end (E2E) and system integration test suites that verify communication flows between contracts, APIs, databases, and client web dashboards.

---

## 1. Directory Structure

```
tests/
├── e2e/                   # E2E system flows & validation test suites (upcoming)
├── integration/           # Cross-service integrations test suites (upcoming)
└── README.md              # Test execution playbook reference
```

---

## 2. Test Execution

System tests run automatically during GitHub CI pipelines on every pull request. To execute integration tests locally, refer to the testing checklists in the core roadmap spec [10-roadmap.md](file:///Users/apple/Documents/UnifyVault-UV/docs/10-roadmap.md).
